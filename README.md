# TODO
This project was created as part of a Python course assignment.This project is a terminal-based Task (TODO) Manager written in Python.
The purpose of the project is to demonstrate fundamental Python knowledge including:

Variables and data types

Conditional statements

Loops

Functions

Object-Oriented Programming (classes)

Modular program structure

File handling

Use of standard and external libraries

Basic unit testing

The program allows the user to:

View tasks

Add new tasks

Mark tasks as completed

Save and load tasks from a file

All functionality is explained through clear comments inside the code, making the project easy to understand and maintain.
